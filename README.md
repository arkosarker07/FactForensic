# FactForensic
Political bias leaning and objectivity score detection from news article system

## Features:
- Classifies bias across 3 categories (Left, Center, Right) using ML.
- Give objectivity score (1-100%) to a news article using llama-3.1-8b-instant.
- Aggregates top news by advanced scraping with real-time updates.
- Breaking News of world and Bangladesh from last 30 minutes.
- Utilized Grmini 2.5 flash API to generate concise summaries, improving information consumption speed
 for users.
